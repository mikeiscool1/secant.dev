It has been rewritten twice, and all three versions are available as separate branches on GitHub. However, the previous versions no longer work due to being too outdated.

Vitchat

Vitchat is an online chatting platform. It has only one global chat. Users need to be whitelisted by an administrator to join. Unfortunately, vitchat doesn't support uploading files, which includes images; however, users can still set a profile picture. Vitchat works on both desktop and mobile devices. In addition to the website, I also created a very simple bot for Vitchat called Barry Bot (visible in the image above). Click here to view its source code.

Snake

This is a reimplementation of Snake using the Java Swing library. Previously, I had written this in C++ with SDL2, but I rewrote it in Java for both the learning experience, as well as to make the application more accessible.

Tetris

This is a reimplementation of the popular retro game Tetris using the SDL2 C graphics library. As the player advances through the game, the speed that the blocks fall increases. The player score is displayed in the top left corner.

Flappy Bird

This is a reimplementation of Flappy Bird using the SDL2 C graphics library. It was one of my first SDL2 projects, and consequently, it is not performant.

Tic-Tac-Toe (Solved)

Tic Tac Toe is a simple game, and therefore it is easy to solve. This program solves the entire Tic Tac Toe game and then allows you to play against it in the terminal. It is impossible to win against this AI; you can only manage to tie with it. This program demonstrates use of the minimax algorithm to calculate the best moves.

GitHub
Lexer

This program converts code from text to lexical tokens. It uses custom syntax for a programming language that does not exist, but the syntax can easily be changed. Originally, this was going to be a programming language, but it only made it to the lexer state. It allows you to print out the tokens with color, essentially also acting like a syntax highlighter.

About Me

More about my programming experience and preferences.

Programming Languages

I have extensive experience with TypeScript and C++, utilizing TypeScript/JavaScript for scripting and networking tasks, and C++ for algorithm-focused or CPU-intensive applications. Recently, I have developed an interest in Rust due to its modern design, powerful features, safety, and high-level abstractions. Additionally, I have limited experience with Python, Java, Assembly, and several other programming languages.

Websites

All my websites are powered by a VPS running on Ubuntu Linux where I use Nginx as the web server and ExpressJS for backend functionality. I rely on Cloudflare for SSL certificates, DNS management, and security. For frontend design, I utilize Bootstrap, a CSS library, to create clean and responsive layouts.

For purchasing domains, I use https://namecheap.com as it offers cheap domains and easy domain management.

Databases

I previously worked with MongoDB, a NoSQL database, but have since transitioned to PostgreSQL, a robust and versatile SQL database. PostgreSQL's advanced features, such as support for complex queries and extensive extensions, have made it my preferred choice for managing relational data. Therefore, it has been the go-to database for all of my projects.
